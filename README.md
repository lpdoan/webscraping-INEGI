# Retrieving data from the INEGI website using Python 3.x
This project uses Selenium in Python 3.x to retrieve the Consumer Price Index data from INEGI, the National Institute of Statistics and Geography of the Mexican government. 

The data is updated monthly with one month lag, featuring historical data of 3 most recent months.

The target website is here : [INEGI MX](https://www.inegi.org.mx/app/indicesdeprecios/Estructura.aspx?idEstructura=112001300020&T=%C3%8Dndices+de+Precios+al+Consumidor&ST=Principales+%C3%ADndices+(mensual)?enablejsapi=1)
